🏢 PropMap
상업용 부동산 거래 정보를 지도 위에 기록하고
팀과 함께 시장 흐름을 분석하는 인텔리전스 플랫폼
✨ 주요 기능
📍
지도 기반 자산 관리
서울 주요 상업용 부동산을 지도 위에 핀으로 등록하고 권역별로 시각화
🕐
매각 히스토리
과거 거래 내역(연도·매도자·매수자·금액)을 타임라인으로 기록
📋
테이블 뷰
전체 자산을 표 형태로 확인, 정렬·필터·CSV 내보내기 지원
☁️
실시간 팀 공유
Firebase 기반 실시간 동기화. 팀원이 추가한 정보가 즉시 반영
📝
메모 & 태그
자산별 메모에 #태그를 달아 주제별로 모아보기 가능
🗞
뉴스 링크
관련 뉴스 URL을 등록하면 헤드라인과 파비콘이 자동 표시
🗺 권역 분류
GBD — 강남 (테헤란로, 역삼, 삼성) YBD — 여의도 CBD — 도심 (종로, 을지로) 성수동
